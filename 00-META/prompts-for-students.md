# Prompts sugeridos para estudiantes

Coleccion de prompts listos para usar con una IA como tutor dentro de este repositorio.

## Tutor paso a paso
```
Actua como tutor de Diseno Digital. Lee el archivo [RUTA-DEL-ARCHIVO]. Explica el concepto principal en 5 frases y dame un ejemplo corto.
```

## Verificador de soluciones
```
Actua como verificador. Compara mi solucion con el archivo [RUTA-DEL-PROBLEMA] y la solucion en [RUTA-DE-SOLUCION-NIVEL-3]. Indica discrepancias concretas y si el resultado final coincide.
```

## Generador de pistas (sin spoilers)
```
Dame solo pistas para el problema en [RUTA-DEL-PROBLEMA]. No reveles la solucion completa; limita la respuesta a 3 sugerencias.
```

## Simplificador de notacion
```
Lee el fragmento [PEGAR TEXTO]. Reescribelo usando la notacion estandar definida en 00-META/notation-cheatsheet.md.
```

## Plan de estudio personalizado
```
Quiero estudiar el subtema [NOMBRE]. Genera un plan de 5 pasos usando Intro, Resumen-Formulas, theory, methods y problems en ese orden. Incluye referencias a archivos.
```

## Generador de autoevaluacion
```
Crea 5 preguntas de autoevaluacion tipo verdadero/falso basadas en [RUTA-DEL-RESUMEN]. Dame las respuestas en una tabla Nivel 2.
```
